# Header & Surface: Blur Layer Refactoring

## Problem Analysis

In SwiftUI 26, headers (navigation bars, toolbars, section headers) have an **automatic blur effect** underneath:
- The blur covers ALL stacked headers as **one continuous region**
- When you scroll and headers stack, the blur region **merges** seamlessly
- No manual wiring â€” it's declarative via `GlassEffectContainer`

Current implementation problems:
1. **`SwiftPinnedHeaderChrome` renders a RED BOX** (`ColoredBox(color: Colors.red.withAlpha(120))`) â€” not actual blur
2. **Blur is disconnected** â€” `SwiftProgressiveBlurSliver` must be manually placed by user, reads `animatedHeight` from controller
3. **Manual wiring required** â€” users must pass `chromeController` via Builder, check for null, pass it to slivers
4. **Each header must explicitly register** â€” `SwiftPinnedHeaderChromeSliver` with explicit `chromeController` parameter
5. **10+ files** with overlapping responsibilities: chrome controller, chrome sliver, progressive blur, backdrop filter, inspire blur sliver

## Implementation Plan

### Phase 1: Create `GlassEffectContainer`

**New file**: `lib/src/materials/glass_effect_container.dart`

```dart
class GlassEffectContainer extends StatefulWidget {
  final Widget child;
}

class GlassEffectContainerState extends State<GlassEffectContainer> {
  // Tracks all registered glass regions
  // Aggregates totalHeight from all visible headers
  // Notifies listeners when entries change
  
  void register(GlassEntry entry);
  void unregister(Object id);
  double get totalHeight; // sum of all visible header heights
  List<GlassEntry> get entries;
}
```

GlassEntry:
```dart
class GlassEntry {
  final Object id;
  final double height;
  final double visibleHeight; // 0 when scrolled away
  final LayerLink link;
}
```

### Phase 2: Create `GlassEffect` modifier

**New file**: `lib/src/materials/glass_effect.dart`

A widget that:
1. Wraps content in glass material (blur + tint)
2. Auto-registers with nearest `GlassEffectContainer`
3. Reports its visible height based on scroll position
4. Can be applied to any widget: `.glassEffect()`

```dart
class GlassEffect extends StatefulWidget {
  final Widget child;
  final double height;
  final GlassEffectStyle style; // regular, prominent, tinted
  final Color? tintColor;
  final Shape shape;
}
```

### Phase 3: Create `GlassEffectBlurSliver`

**New file**: `lib/src/materials/glass_effect_blur_sliver.dart`

Replaces `SwiftProgressiveBlurSliver`. This sliver:
1. Auto-discovers `GlassEffectContainer` via context
2. Reads all registered entries
3. Renders one continuous blur region covering all visible headers
4. Uses the existing `inspire_blur` shaders

```dart
class GlassEffectBlurSliver extends StatefulWidget {
  final double fadeLength;
  final double maxBlurSigma;
}
```
